VERIFICATION

This package wraps the official kanban Windows release binaries
(`kanban.exe` and `kanban-mcp.exe`), which are downloaded at install
time from the project's GitHub release page as a single ZIP archive:

  https://github.com/fulsomenko/kanban/releases

The SHA256 checksum of the downloaded archive is embedded in
chocolateyinstall.ps1 via the `$checksum64` substitution variable,
populated at pack time by downloading the GitHub release ZIP and
running SHA256 over it. The same value is also published in the
`SHA256SUMS` file attached to the corresponding release; the two
are independently computed from the same archive.

To verify the binaries manually:

  1. Download the release ZIP from:
       https://github.com/fulsomenko/kanban/releases/download/v<VERSION>/kanban-v<VERSION>-x86_64-pc-windows-msvc.zip
  2. Compute its SHA256:
       Get-FileHash -Algorithm SHA256 <file>.zip
  3. Compare the computed SHA256 against the checksum displayed on
     the package's chocolatey.org page (under the "Files" tab):
       https://community.chocolatey.org/packages/kanban
     The page surfaces the embedded `checksum64` value from
     chocolateyinstall.ps1.
  4. Unzip and confirm the archive contains `kanban.exe`,
     `kanban-mcp.exe`, `LICENSE.md`, and `README.md`.

The source for the binaries is at:
  https://github.com/fulsomenko/kanban

The build is produced by the project's release.yml workflow against
the tagged commit. The same source produces functionally equivalent
binaries; bit-exact reproducibility is not currently a project goal
(Compress-Archive is non-deterministic across runs and cargo embeds
some build metadata).
